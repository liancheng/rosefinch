Sphinx主题——朱雀
================

“朱雀（rosefinch）”是一款基于\ `Twitter Bootstrap`__\ 的\ `Sphinx`__\ 主题。通过集成由\ `@ethantw`__\ 开发的“\ `汉字标准格式`__\ ”CSS框架，“朱雀”优化了汉字的网页显示效果。

.. note::

   本主题尚在开发中，暂不推荐使用。

.. compound::

   “朱雀”这个名字取自该主题的三个要素：

   *   希腊神话怪物斯芬克斯（Sphinx）
   *   鸟（Twitter）
   *   中文文档优化
